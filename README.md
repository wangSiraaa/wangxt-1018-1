# 药企展厅访客预约系统

面向药企展厅的市场接待全流程访客预约系统，覆盖市场批次创建、访客预约填报、接待签到、涉密审批、候补队列、路线安排、安保名单、离场核销、审计日志九大模块。

## 快速开始

### 方式一：脚本启动（推荐）

```bash
chmod +x start.sh
# 开发模式
./start.sh dev
# 生产构建 + 预览
./start.sh build && ./start.sh preview
# Docker 一键启动
./start.sh docker
```

### 方式二：手动命令

```bash
# 安装依赖
npm install
# 开发模式：http://localhost:5173
npm run dev
# 生产构建
npm run build
# 预览：http://localhost:4173
npm run preview
```

### 方式三：容器发布

```bash
docker compose up -d --build
# 访问 http://localhost:4173
```

## 本地数据预置

- 批次：3个（常规/涉密/VIP，含不同路线）
- 预约：7条（覆盖签到中、候补中、待NDA、已离场等状态）
- 路线：3条（含涉密展区NDA）
- 黑名单：2家（用于预约拦截演示）

## 四大演示场景操作指引

### 1. 人数超限候补
1. 进入 `/waiting` 或首页「人数超限候补」卡片
2. 观察 6月14日下午涉密批次 容量已满（8/8），2家企业在候补
3. 点击「模拟取消释放名额」→ 自动弹出转正提示

### 2. 涉密未确认拦截
1. 进入 `/reception` 接待签到台
2. 切换到「今日待签到」，找到「康美生物科技」（涉密标签）
3. 点击签到按钮 → 弹出红色错误：NDA未完成无法签到
4. 跳转 `/secret` 完成审批 → 再回来即可签到

### 3. 签到后不能取消
1. 在 `/reception` 将任意预约完成签到
2. 打开「批次详情」或「预约列表」尝试取消
3. 弹出提示：已签到访客禁止取消，请走离场核销流程
4. 进入 `/checkout` 完成离场

### 4. 路线冲突调整
1. 进入 `/routes` 或首页「路线冲突调整」卡片
2. 创建新批次：选择 6月14日 09:30-11:00，讲解员王丽、路线A
3. 顶部弹出红色冲突警告：讲解员在该时段已有安排
4. 更换为其他讲解员或时间 → 冲突解除
